
# 美团搜索，获取美团店铺数据，逆向美团token参数

## 本篇美团有三种获取 _token方式
### 1、纯算法实现，使用python的zlip模块，对数据进行二次压缩加密
### 2、纯逆向实现，直接扣JS代码，并修改部分JS代码
### 3、使用RCP获取加密函数，对美团网站加密位置注入JS代码，持久化保存加密函数后调用该加密函数
